TorBox Streamer Settings

Get your API key from torbox.app/settings.
Change this if the default Torrentio instance is blocked by Cloudflare.

🍿 Optional: Launch directly in MPV

To launch streams in external MPV player directly, run the one-line setup helper script on your system:

python3 helpers/install.py No separate desktop software required. Works out of the box with python3 & mpv.
